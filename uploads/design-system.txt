[Heading1] Introduction
This document specifies the complete visual system for silvi.haldipur.com — a personal website for a senior pharma commercial executive with twenty years of experience installing new capabilities inside Fortune 500 organizations.
The visual register is editorial, not corporate. The reference points are Cereal magazine, Aesop, The Row, Le Labo, and FT Weekend supplements — not SaaS landing pages or typical executive personal-brand sites. The audience is Chief Commercial Officers, Chief Experience Officers, and executive recruiters at top-twenty pharma. They pattern-match on visual restraint. Every design decision should signal &quot;next CXO&quot; rather than &quot;consultant for hire.&quot;
This document specifies five pages: Home, About, Work, Perspectives, and Contact. It begins with the global design tokens (color, typography, layout primitives, components) that apply across every page, then specifies each page in detail. The order matters: read the global system first, then the page-level instructions inherit from it.
All copy is drawn from Silvi_Haldipur_website_copy_v9_1.docx. The &quot;Change notes&quot; sections at the bottom of that document are internal editorial notes and must not appear on the live site.

[Heading2] Color palette
Use these as CSS variables. Do not introduce any color outside this palette.
:root {
  --cream: #F5EDE2;          /* page background */
  --oxblood: #78281F;        /* primary brand: headlines, CTAs, links */
  --oxblood-deep: #5C1E17;   /* CTA hover state */
  --sienna: #B8593A;         /* hero portrait field, large warm blocks */
  --coral-sienna: #C9633F;   /* answer-lead borders and accent italics */
  --coral-light: #E89472;    /* italic emphasis on dark grounds */
  --ink: #2B1810;            /* dark band ground, body type */
  --ink-soft: #4A2B1F;       /* secondary body text on cream */
  --rule: rgba(120, 40, 31, 0.15);  /* hairline dividers */
}
Rules: never use pure white or pure black anywhere. Sienna and oxblood are the same warm family and belong on the same page. Do not introduce a third unrelated accent. Text on sienna or oxblood grounds is always cream. Text on cream is ink or ink-soft, with oxblood reserved for headlines, links, and emphasis.

Two families only. Sentence case throughout — never Title Case headlines, never ALL CAPS except for small eyebrow labels and nav links.
